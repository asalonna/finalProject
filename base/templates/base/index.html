Website Guide


Classrooms

Click the ‘Create classroom’ link on the navigation bar.

  1. Name your classroom in the first field. The name has a limit of 100 characters.
  2. The questions in your classroom will need to be given different difficulty levels. In the second text field, you can choose how many questions at a given level need to be correctly answered in order to progress onto the next difficulty.
  3. Enter a numerical passcode for your classroom. This passcode will be used to track grades.
  4. Once you submit the classroom creation form, an access code for it will appear in the label beneath the submit button.


Creating questions

Click the ‘Create questions’ link on the navigation bar.

  1. Enter a name for your question. Question names are limited to 100 characters.
  2. Write your question using the DSL.
  3. Optionally, you can use the DSL to specify restrictions for your question.
  4. Use the DSL to specify the expected answer for your question.
  5. Select a difficulty level for your question.
  6. Choose how many attempts are allowed for your question.
  7. Enter the access code generated in the classroom creation page to assign this question to a class.
  8. After you submit the form, you will receive an access code for your question. (This can be used to alter/delete??)


Accessing classrooms (and the questions within)

Students need to click the ‘Login’ link on the navigation page to be able to access the questions.

  1. Students should enter their student ID’s here.
  2. The classroom access code that was generated after the creation of a classroom can be given to students so they can access the questions.


Answering Questions

After logging in, students will be directed to the question answering page. Students will use the Code Editor to write their answers. The Output box will show the output of their program, as well as error messages if something is amiss. The Feedback box will display whether or not they have successfully completed the question.

If the answer is correct, a Next Question button will appear beneath the Submit Work button. This will take the student to the next question.


Track Questions

  1. Enter the classroom access code generated when the classroom was created.
  2. Enter the password for the classroom.
  3. When you press submit, you will be able to see a Student Summary: This will show you how many students have attempted any question in the class, how many completed the class, and the average highest difficulty reached by students.
  4. Classroom Data shows for each user who has accessed the questions whether they completed the class or not, as well as the highest difficulty of the questions they have done successfully.
